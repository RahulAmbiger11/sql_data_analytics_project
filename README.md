# SQL Data Analytics Project

This repository is a practical collection of SQL scripts built for data exploration, analytics, and reporting. It provides ready-to-use queries designed to help data analysts and BI professionals quickly understand, segment, and extract insights from a relational database.

---

## 📦 What's Inside

The scripts are organized by analytical themes, featuring clean code for:

* **Database Exploration:** Quickly understanding table structures and data distributions.
* **Metrics & Measures:** Calculating key performance indicators and core business numbers.
* **Time-Based Trends:** Analyzing data changes, growth, and patterns over days, months, or years.
* **Cumulative Analytics:** Running running totals, moving averages, and year-over-year growth.
* **Data Segmentation:** Grouping and profiling data to find specific behaviors or trends.
